# GCP Cloud Security Auditor

## Description
This Python-based tool automates security audits for Google Cloud Platform (GCP) storage buckets. It identifies public access vulnerabilities, ensuring alignment with **ISO 27001** access control standards.

## Key Features
* **Automated Scanning:** Lists all buckets in the GCP project.
* **Vulnerability Detection:** Identifies "allUsers" or "allAuthenticatedUsers" permissions that expose data.
* **Compliance Focus:** Designed for GRC (Governance, Risk, and Compliance) workflows.

## Technologies Used
* **Language:** Python
* **Cloud Platform:** Google Cloud (GCP)
* **Libraries:** google-cloud-storage
